# 오라클 Blockchain Workshop

## 소개 ##
이 워크샵은 Oracle 블록체인 플랫폼을 통해 환경을 구성하고 직접 블록체인 코드의 개발과 배포를 경험해 볼 수 있는 Hands-on 워크샵입니다.
블록체인 네트워크의 구성과 스마트컨트랙트를 개발 해봄으로써 블록체인이 기업환경에서 어떤 식으로 적용할 수 있을 지를 체험할 수 있게 됩니다.
